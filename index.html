Trabalhando com formulario

Início

Tags padrão:

  • <h1> até <h6>: Criam cabeçalhos ou títulos com tamanhos variados, sendo <h1> o maior e <h6> o menor.
  • <p>: Define parágrafos, organizando o texto em blocos.
  • <mark>: Destaca o texto com uma cor de fundo.
  • <small>: Reduz o tamanho da fonte para textos menos importantes.
  • <i>: Aplica o estilo itálico ao texto.
  • <u>: Sublinha o texto.
  • <strong>: Dá ênfase ao texto, exibindo-o em negrito.
  • <ol>: Cria uma lista ordenada com numeração.
  • <ul>: Cria uma lista não ordenada com marcadores.
  • <li>: Representa itens dentro de listas (ordenadas ou não).
  • <a>: Cria links para outras páginas ou recursos. O atributo href define o destino.
  • <hr>: Insere uma linha horizontal para separar o conteúdo.
  • <sub>: Exibe texto em subscrito, abaixo da linha base.
  • <sup>: Exibe texto em sobrescrito, acima da linha base.
  • <blockquote>: Insere uma citação em bloco, geralmente com recuo.

Tags Sugeridas

  • <font>: Estiliza o texto com atributos como cor, tamanho e fonte (considerada obsoleta em HTML5).
  • <del>: Risca o texto, indicando remoção ou alteração.
  • <abbr>: Define abreviações e exibe o significado completo com o atributo title ao passar o mouse.

Estruturando seu HTML + formatação

Início

Formatando Textos - Parte 1: Ensina o uso de tags básicas para estruturar e estilizar textos, como <h1> a <h6>, <p>, <mark>, <small>, <strong>, e <i>, garantindo formatações claras.
Formatando Textos - Parte 2: Expande o aprendizado com listas ordenadas (<ol>) e não ordenadas (<ul>), além de links (<a>) e outras ferramentas de estrutura.
Div e Span: A tag <div> organiza o layout agrupando elementos em blocos, enquanto <span> permite estilizar pequenas partes de texto.
Fieldsets: Usada para agrupar campos de formulários, junto com a tag <legend> para descrever o grupo, melhorando a organização.
Embeds: A tag <embed> permite incorporar conteúdo multimídia externo, como vídeos, diretamente na página HTML.
Iframes: Cria uma área embutida para carregar outro documento HTML ou páginas da web dentro da página principal.
Resenha Sobre Cores: As cores podem ser definidas em HTML usando nomes de cores, valores RGB, hexadecimal ou HSL. Aplicam-se diretamente nos elementos por meio de atributos como style="color" ou com CSS externo.

Trabalhando com mídias ultilizando HTML

Início

Formatando Textos - Parte 1: Explica como usar tags básicas como <h1> a <h6>, <p>, <strong>, <i>, <u>, e <mark> para estruturar e estilizar textos.
Formatando Textos - Parte 2: Complementa a formatação abordando listas ordenadas (<ol>) e não ordenadas (<ul>), além de links (<a>) para interatividade.
Div e Span: A tag <div> é usada para agrupar elementos em blocos, enquanto <span> estiliza pequenas partes de texto dentro de um bloco maior.
Fieldsets: Usado para organizar elementos de formulários, agrupando campos com a tag <fieldset> e adicionando descrições com <legend>.
Embeds: Permite incorporar arquivos multimídia, como vídeos, diretamente na página HTML usando <embed>.
Iframes: Insere páginas da web ou conteúdos externos dentro da página principal por meio da tag <iframe>.
Resenha Sobre Cores: Define as cores em HTML utilizando valores RGB, hexadecimal, HSL ou nomes de cores. Pode ser aplicado diretamente nos elementos com o atributo style ou através de CSS.

Criando tabelas

Início

Tag table: Utilizada para criar tabelas em HTML. A tag <table> define a estrutura da tabela.
Tag tr: Representa uma linha na tabela. Cada linha da tabela é definida pela tag <tr>.
Tag td e th: A tag <td> define células de dados (conteúdo regular) e <th> define células de cabeçalho da tabela, geralmente exibindo texto em negrito e alinhado ao centro.
Tag tbody, thead e tfoot: <thead> define o cabeçalho da tabela, <tbody> agrupa as linhas de conteúdo, e <tfoot> representa o rodapé da tabela.
Estilizando Tabelas: Para estilizar tabelas, é possível usar atributos HTML, como border e cellspacing, ou aplicar estilos via CSS para ajustar bordas, cores, espaçamento e alinhamento.
